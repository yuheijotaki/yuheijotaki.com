---
name: explain-code
description: コードを視覚的な図とアナロジーで説明する。コードの動作を説明するとき、「これはどう動くの？」と聞かれたときに使用。
---

コードを説明するときは、必ず以下を含めてください：

1. **アナロジーから始める**: コードを日常生活の何かに例える
2. **図を描く**: ASCII アートでフロー、構造、関係性を示す
3. **コードをウォークスルーする**: 何が起こるかステップバイステップで説明
4. **注意点を強調する**: よくある間違いや誤解は何か？

説明は会話調で。複雑な概念には複数のアナロジーを使用。
