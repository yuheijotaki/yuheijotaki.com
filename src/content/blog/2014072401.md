こっからいじっても問題ないはず

<ul>
<li>.replaceは | で区切って検索文字複数可</li>
</ul>

```js
var str = 'abcdefg';
alert(str.replace(/abc|efg/g,'xxx')); //'xxxcxxx'
```

<ul>
<li>.sliceは2つ目の引数省略で最後まで</li>
</ul>
これ良く使う

```js
var str = 'abcdefg'
if (str.substring(0,1)=='a') {
	alert(str.slice(1)); //'bcdefg'
}
```
