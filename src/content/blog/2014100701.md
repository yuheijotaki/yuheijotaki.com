◯導入  
（Rubyインストール）

<ul>
<li>ターミナルで</li>
</ul>

```html
(sudo) gem install middleman
```

<ul>
<li>プロジェクト作成（desktop/testの場合）</li>
</ul>

```html
cd desktop/test
middleman init .
```

<ul>
<li>Middlemanの起動</li>
</ul>

```html
middleman server
```

<ul>
<li>起動したサーバのURLを確認し、ブラウザでアクセス</li>
</ul>

```html
The Middleman is standing watch at http://0.0.0.0:4567
```

<ul>
<li>適当にいじる</li>
</ul>

<ul>
<li>Middlemanの終了</li>
</ul>
ターミナルで「Ctrl+C」

<ul>
<li>静的ファイルの書き出し（ビルド）</li>
</ul>

```html
middleman build
```
