middleman-blogをつくる

gem “middleman-blog” とか前回の引き継いだらうまく行かなかったので新規でデスクトップに「blog」フォルダを作成

ターミナルで移動

```html
cd desktop/blog
```

Middleman Blog Extentionと一緒にインストール

```html
middleman init myproject --template=blog
```

たくさんファイルが追加されてます

Middleman起動

```html
middleman
```

ブラウザで確認

できた

config.rbにて設定などするのは今度します
