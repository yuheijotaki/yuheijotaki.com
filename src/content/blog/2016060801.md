スライダーの幅500pxの場合

```html
<ul class="slide">
	<li><img src="..."></li>
	<li><img src="..."></li>
</ul>
```

```css
ul.slide {
	width: 500px;
	overflow: hidden;
	margin: 0 auto;
}
```

```js
$('.slide').bxSlider({
	slideWidth: 500
});
```
