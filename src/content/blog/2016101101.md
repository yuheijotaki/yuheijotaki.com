Vagrant と WordMove を使った案件。  
どちらも初めてで、Gitに関しても実案件がっつりは初なので、  
実装どうこうより開発環境を使う &amp; 慣れるのが大変でした。

そもそもターミナルの操作  
◯処理中で終了

```html
ctr c
```

◯ターミナルでvim

```html
i で編集
esc で編集終了
:wq 保存して終了
:q! 保存せずに終了
```

Vagrant  
◯よく使う  
Vagrant起動

```html
$sudo vagrant up
```

Vagrant状況確認

```html
$sudo vagrant status
```

Vagrant再起動

```html
$sudo vagrant reload
```

仮想マシンへ接続

```html
$sudo vagrant ssh
```

WordMove

```html
cd /vagrant && wordmove push -t -e hoge
```

「-t」はオプションでテーマファイル

```html
-u
uploads

-t
wp-content/theme

-p
wp-content/plugins

-d
DB

--all
すべてまるまる
```
