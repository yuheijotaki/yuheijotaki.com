<h3>読み込み</h3>

CDN の場合は<a href="https://jp.vuejs.org/v2/guide/installation.html#CDN">公式</a>から最新バージョン

```html
<script src="https://cdn.jsdelivr.net/npm/vue@2.5.21/dist/vue.js"></script>
```

<h3>Hello world</h3>

```html
<div id="app"><p>{{ message }}</p></div>
```

```js
var app = new Vue({
  el: "#app",
  data: {
    message: "Hello world"
  }
});
```

コンソール

```js
console.log(app.message); // Hello world
```

<h3>オブジェクトから</h3>

```html
<div id="app"><p>{{ item.message }}</p></div>
```

```js
var app = new Vue({
  el: "#app",
  data: {
    item: {
      message: "Hello world"
    }
  }
});
```

下記にすると文字数を出力

```html
<div id="app"><p>{{ item.message.length }}</p></div>
```

<h3>ループ（リスト）</h3>

```html
<div id="app">
  <ol>
    <li v-for="item in list">{{ item }}</li>
  </ol>
</div>
```

```js
var app = new Vue({
  el: "#app",
  data: {
    list: ["りんご", "ばなな", "いちご"]
  }
});
```

下記にすると<code>りんご</code>を出力

```html
<div id="app"><p>{{ list[0] }}</p></div>
```

配列への追加

```js
app.list.push("おれんじ");
```

<h3>ループ（オブジェクトから）</h3>

```html
<div id="app">
  <ul>
    <li v-for="item in list">
      <p>ID:{{ item.id }}</p>
      <p>NAME:{{ item.name }}</p>
      <p>PRICE:{{ item.price }}</p>
    </li>
  </ul>
</div>
```

```js
var app = new Vue({
  el: "#app",
  data: {
    list: [
      { id: 1, name: "りんご", price: 100 },
      { id: 2, name: "ばなな", price: 200 },
      { id: 3, name: "いちご", price: 300 }
    ]
  }
});
```

下記にすると<code>1</code>〜<code>15</code>まで連番で出力

```html
<div id="app">
  <p><span v-for="item in 15">{{ item }}</span></p>
</div>
```

下記にすると<code>1,5,10,15</code>が出力

```html
<div id="app">
  <p><span v-for="item in [1,5,10,15]">{{ item }}</span></p>
</div>
```

```html
<div id="app">
  <p><input type="text" v-bind:value="message" /></p>

```

```js
var app = new Vue({
  el: "#app",
  data: {
    message: "Hello world"
  }

```

<h3>クラスのバインディング</h3>

クラス名にハイフン、アンダースコアがある場合は<code>'</code>で囲う

```html
<div id="app">
  <p v-bind:class="{isActive:isActive,'js-active':jsActive}">りんご</p>
</div>
```

```js
var app = new Vue({
  el: "#app",
  data: {
    isActive: true,
    jsActive: true
  }
});
```

オブジェクトでも可能

```html
<div id="app"><p v-bind:class="classObject">りんご</p></div>
```

```js
var app = new Vue({
  el: "#app",
  data: {
    classObject: {
      isActive: true,
      "js-active": true
    }
  }
});
```

<h3>条件分岐（<code>v-if</code>,<code>v-show</code>）</h3>

```html
<div id="app">
  <div v-if="ok">`ok: true`の場合出力される</div>
  <div v-show="ok">ok: true`の場合出力される</div>
</div>
```

```js
var app = new Vue({
  el: "#app",
  data: {
    ok: true
  }
});
```

<a href="https://jp.vuejs.org/v2/guide/conditional.html#v-if-vs-v-show">v-if vs v-show</a>

<blockquote>一般的に、v-if はより高い切り替えコストを持っているのに対して、 v-show はより高い初期描画コストを持っています。 そのため、とても頻繁に何かを切り替える必要があれば v-show を選び、条件が実行時に変更することがほとんどない場合は、v-if を選びます。

</blockquote>

<h3>条件分岐（<code>&lt;template&gt;</code>）</h3>

```html
<div id="app">
  <template v-if="ok">
    <div>
      <h1>Vue.js</h1>
      <p>Hello world</p>
    </div>
  </template>
</div>
```

```js
var app = new Vue({
  el: "#app",
  data: {
    ok: true
  }
});
```

<h3>条件分岐（<code>v-if</code>,<code>v-else-if</code>,<code>v-else</code>,）</h3>

```html
<div id="app">
  <div v-if="type === 'A'"><p>`type: 'A'`の場合出力される</p></div>
  <div v-else-if="type === 'B'"><p>`type: 'B'`の場合出力される</p></div>
  <div v-else><p>すべての条件を満たさなかった場合出力される</p></div>
</div>
```

```js
var app = new Vue({
  el: "#app",
  data: {
    type: "A"
  }
});
```
