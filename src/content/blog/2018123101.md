スニペットは Sublime Text の場合.xml、vscode は.json なので、変換ツール <a href="https://www.npmjs.com/package/convert-snippets-to-vscode">convert-snippets-to-vscode</a> を使いました。

<h4>手順</h4>

<a href="https://vscode-doc-jp.github.io/docs/userguide/userdefinedsnippets.html">独自のスニペットを作成
</a> の通り <code>"editor.tabCompletion": true</code> に。

コマンドから convert-snippets-to-vscode をインストール

```sh
npm install -g convert-snippets-to-vscode
```

デスクトップに移動して、あらかじめ html/scss/javascript/php など言語ごとに Sublime のスニペットをフォルダ分けしておく。

```
snippetToVsCode
```

とすると

```
? Folder name: html
```

でフォルダ名（html など）

```
? Output File Name: html.json
```

で出力するファイル名（html.json など）を入力。
