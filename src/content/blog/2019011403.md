<h3>Chapter2 データを表示するとき</h3>

<h4>01 Vue インスタンスを作る：<code>new Vue</code></h4>

```js
new Vue({
  el: ,       // どのHTML要素とつながるのか
  data: ,     // どんなデータがあるのか
  methods: ,  // どんな処理を行うのか
  computed: , // どのデータを使って別の計算をするのか
  watch:      // どのデータを監視するのか
});
```

<h4>02 データをそのまま表示する：<code>{{データ}}</code></h4>

データバインディング

<blockquote>「Vue インスタンスのデータが Web ページ上の表示と結合すること」を<strong>データバインディング</strong>といいます。

</blockquote>

ディレクティブ

<blockquote><strong>ディレクティブ</strong>とは、「要素に対して Vue がどんなことを行うかを指示する命令」のことで頭に <code>v-</code> がついています。

</blockquote>

<h4>03 使えるデータの種類</h4>

```html
<div id="app">
  <!-- 数値型 -->
  <p>{{ myNumber }}</p>
  <hr />
  <!-- 文字列型 -->
  <p>{{ myText }}</p>
  <hr />
  <!-- ブーリアン型 -->
  <p>{{ myBool }}</p>
  <hr />
  <!-- 配列 -->
  <p>{{ myArray }}</p>
  <p>{{ myArray[0] }}</p>
  <hr />
  <!-- オブジェクト型 -->
  <p>{{ myObject }}</p>
  <p>{{ myObject.name }}：{{ myObject.gender }}</p>
  <hr />
  <!-- JavaScriptで用意したデータ -->
  <p>{{ jsObject }}</p>
  <p>{{ jsObject[1].name }}：{{ jsObject[1].gender }}</p>
  <hr />
  <!-- データの出力 -->
  {{ $data }}
  <li v-for="(item,key) in $data">{{ key }}:{{ item }}</li>
</div>
```

```js
var jsList = [
  { name: "Hanako", gender: "female" },
  { name: "Ichiro", gender: "male" },
  { name: "Tomoko", gender: "female" }
];
new Vue({
  el: "#app",
  data: {
    myNumber: 12345,
    myText: "Hello!!!",
    myBool: true,
    myArray: ["Taro", "Hanako", "Ichiro", "Tomoko"],
    myObject: {
      name: "Taro",
      gender: "male"
    },
    jsObject: jsList
  }
});
```
