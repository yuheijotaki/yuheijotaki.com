Vue CLI を使って JSON から情報を引っ張る。  
とりあえず WordPress の <code>/wp-json/</code> から。

scssを有効化

```sh

```

<li><a href="https://qiita.com/MariMurotani/items/5fbea5942d2edf149989">VueCLIでHelloWorld - Qiita</a></li>

<h3>リセットCSS（normalize.css）</h3>

```sh

```


<h2>WordPress の /wp-json/</h2>

知らなかったのですが、以前までパラメータ与えると投稿をフィルタして取得できたのですが、4.7から仕様が変わったようです。

<blockquote>Wordpress4.7からfilterパラメーターは削除されているので、プラグインを利用してfilterパラメーターを利用できるようにする
<a href="http://umadash.hatenadiary.jp/entry/2019/01/09/093554">WP REST API v2を利用し、カスタム投稿タイプの記事をカスタムタクソノミーでフィルタリングして取得する - エンジニアうまの日記</a>

</blockquote>

とりあえず JSON であればなんでも良いので <a href="https://blog.yuheijotaki.com/wp-json/">https://blog.yuheijotaki.com/wp-json/</a> から引っ張る

<h2>App.vue</h2>

```html
<div id="app">
  <ul>
    <li>name: {{results.name}}</li>
    <li>home: {{results.home}}</li>
    <li>timezone_string: {{results.timezone_string}}</li>
  </ul>
</div>
```

```js
// normalize.css を読み込む
import "normalize.css";
// Ajax通信ライブラリ
import axios from "axios";
// JSON の URL
const jsonUrl = "https://blog.yuheijotaki.com/wp-json/";

export default {
  name: "App",
  data() {
    return {
      results: []
    };
  },
  mounted() {
    // JSON取得
    axios.get(jsonUrl).then(response => {
      this.results = response.data;
    });
  }
};
```
