<h4>HTML5 History モード</h4>

今回は <a href="https://router.vuejs.org/ja/guide/essentials/history-mode.html#%E3%82%B5%E3%83%BC%E3%83%90%E3%83%BC%E3%81%AE%E8%A8%AD%E5%AE%9A%E4%BE%8B">HTML5 History モード | Vue Router</a> を使用してみた。  
<code>/router/index.js</code> で <code>mode: 'history'</code> の指定を追加。  
参考：<a href="https://teratail.com/questions/112717">Vue.js - vue-routerの、hashモードと、historyモードの役割の違いについて｜teratail</a>

```js
export default new Router({
  mode: 'history',
  routes: [
    {
      path: "/",
      name: 'top',
      component: top
    },
    {
      path: "/about",
      name: 'about',
      component: about
    }
  ]
})
```

<h4>vue-router 使用時の現在地ナビ（カレント表示）</h4>

現在いるページのナビゲーション <code>&lt;a&gt;</code> 要素にはデフォルトで <code>.router-link-exact-active</code> というクラスが付与されるが、それを変更したい場合

```html
<ul>
  <li><router-link to="/" exact-active-class="is-selected">Top</router-link></li>
  <li><router-link to="/about" exact-active-class="is-selected">About</router-link></li>
</ul>
```

のように <code>exact-active-class="is-selected"</code> としてあげる

参考：<a href="https://router.vuejs.org/api/#event">API Reference | Vue Router</a>  
参考：<a href="https://qiita.com/kimullaa/items/a75a47f504c75058081f">Vuejs vue-routerはアクティブなリンクに自動でクラスを振ってくれる - Qiita</a>
