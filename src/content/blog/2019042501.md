---
title: '【PHP】変数名に変数の値を代入する'
description: ''
pubDate: 04/25/2019 23:00:00
---


```php
$array = [
  'name01' => 'Taro',
  'name02' => 'Hanako',
  'name03' => 'Jiro'
];
```

という配列があって、

```php
$targetName = 'Hanako';
```

のように、宣言した変数の値 <code>Hanako</code> とオブジェクトの値がマッチする場合を判定し、

```php
$name02 = true;
```

という変数を作りたい場合、

```php
<?php
  $targetName = 'Hanako';
  $array = [
    'name01' => 'Taro',
    'name02' => 'Hanako',
    'name03' => 'Jiro'
  ];
  foreach ($array as $key => $value){
    // 変数 key を宣言
    // オブジェクトの value が $targetName と同じ場合は true にする
    if (strpos($targetName,$value) !== false) {
      ${$key} = true;
    } else {
      ${$key} = false;
    }
  }
  // $name01 = false;
  // $name02 = true;
  // $name03 = false;
?>
```

とする。

<code>${$variableName}</code> と書くことで、変数名に変数の値を代入できる。
