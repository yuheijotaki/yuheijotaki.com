<h3>ファイル構成</h3>

```
/src/
└ App.vue
└ main.js
└ /components/
  └ work-header.vue
  └ work-list.vue
  └ work.vue
```

<code>App.vue</code> ファイルにて <code>axios</code> を使ってJSON取得、 表示判定は <code>work-list.vue</code> で行う。  
<code>work-header.vue</code> からチェックボックス状態のプロパティと値を <code>props</code> して  <code>work-list.vue</code> で表示判定  
<code>work.vue</code> はアイテム1つ単位の表示専用コンポーネント

<h3>いじった所</h3>

WordPress の REST API で取得できるカテゴリーが

```json
[
  0: {
    ...
    category_name: "Front-end ,WordPress"
    ...
  },
  ...
]
```

というオブジェクトになるので、

```js
  methods: {
    request: function(){
      axios.get( 'https://works.yuheijotaki.com/wp-json/wp/v2/posts?per_page=100' )
      .then( response => {
        this.works = response.data; // JSONデータの取得
        let obj = this.works;
        Object.keys(obj).forEach((key) => {
          const catName = obj[key].category_name; // この投稿が属するカテゴリー
          if (catName.includes('Front-end')) {
            obj[key].isCatFrontEnd = true;
          } else {
            obj[key].isCatFrontEnd = false;
          }
  ...
```

のように <code>isCatFrontEnd</code> のプロパティと <code>true</code> or <code>false</code> の値をつけた。  
<code>work-list.vue</code> の <code>computed</code> でこの処理やったほうがスマートそうですが、できるかよく分からずJSONに書き足す形になっちゃいました。
