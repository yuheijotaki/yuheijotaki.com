<h3>Vue.js で変数や mixin を使う</h3>

<a href="https://gootablog.com/vuejs-sass-globalvariables">Vue.jsでSassを使う時にグローバル変数を読み込む方法 - Goota</a> の通りなのですが、 <code>sass-resources-loader</code> を npmインストールして、<code>/build/utils.js</code> のなか、

```js
...
  scss: generateLoaders('sass'),
...
```

を

```js
...
  scss: generateLoaders('sass').concat(
    {
      loader: 'sass-resources-loader',
      options: {
        resources: path.resolve(__dirname, '../src/sass/_base.scss')
      }
    }),
...
```

とする
