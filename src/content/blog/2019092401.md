<h4>sass を使う</h4>

index.vueやcomponents配下の.vueファイルでsassを使うには、ふつうのVueと同じで
<code>$ npm i -D sass-loader node-sass</code>  
をインストールする

<h4>_mixin.scss など .scssのグローバルファイルを使う</h4>

<code>$ npm run dev</code> をすると触れるようになるのですが、Sassのグローバルファイルを使いたい場合、Nuxt Style Resources というモジュールをインストールして nuxt.config.js に設定情報を記述する。  
<code>$ npm i -D @nuxtjs/style-resources</code>  
nuxt.config.js

```js
module.exports = {
  // ...
  modules: ['@nuxtjs/style-resources'],
  styleResources: {
    scss: [
      '~/assets/sass/foundation/_variable.scss',
      '~/assets/sass/foundation/_mixin.scss',
      '~/assets/sass/foundation/_common.scss'
    ]
  }
}
```

これでindex.vueやcomponents配下の.vueファイルで_variable.scssで設定した変数が使えるようになる。
