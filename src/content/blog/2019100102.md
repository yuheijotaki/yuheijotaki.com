インストールされている Node.js のバージョン確認

```sh
$ nodebrew list
v10.15.0
v11.6.0
v12.11.0

current: v12.11.0
```

安定版のインストール

```sh
nodebrew install-binary stable
```

バージョン指定でインストール

```sh
nodebrew install-binary v10.16.3
```

使用するバージョンの切り替え

```sh
$ nodebrew use v10.15.0
use v10.15.0
```

現在のバージョン確認

```sh
$ node -v
v10.15.0
```
