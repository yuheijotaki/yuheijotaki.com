<h2>dataで取得したカラーコードをcssネイティブ変数化</h2>

必要なところのみ

```js
<template lang="pug">
  div(:style="customStyle")
</template>

<script>
export default {
  props:{
    'post': {
      type: Object,
      default: ''
    }
  },
  computed: {
    customStyle () {
      return {
        '--user-color': this.post.customColor
      }
    }
  }
}
</script>

<style lang="scss" scoped>
p {
  color: var(--user-color);
}
</style>
```

参考： <a href="https://shayneo.com/blog/binding-css-variables-with-vue/">Binding CSS Variables in Vue</a>
